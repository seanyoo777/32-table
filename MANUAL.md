## 1. 시스템 전체 구조

```
홈 (/)
├── 랭킹 관리 (/rankings)       선수·페어·단체팀 CRUD, CSV 가져오기/내보내기, 전적 조회
├── 대회·대진표 (/tournament)   토너먼트·리그·조별전, 결과입력, 부전승, CSV 내보내기
├── 경기일정표 (/schedule)      자동 코트 배치, 다일차 지원, CSV/인쇄
├── 점수 입력 (/score)          실시간 스코어보드 + 직접입력, 검증
├── 운영 대시보드 (/dashboard)  LIVE 현황, 경기 호출(콜링), 미확인 기록 관리
├── 체크인 (/checkin)           QR 스캔, 수동 체크인, 출석 CSV
├── 라이브 보드 (/liveboard)    TV/프로젝터 전광판 (전체화면)
├── 통계·리포트 (/stats)        대회 분석: KPI·진행률·메달표·포인트TOP·부문분포
└── 설정 (/settings)           대회 정보, JSON 백업/복원, 데이터 초기화, 다크모드
```

## 2. 대회 전 준비 체크리스트

### 2-1. 초기 설정 (설정 페이지)
1. **설정** → 주최기관명·담당자·시즌·연락처 입력 → **저장**
2. 이전 대회 데이터가 있으면 **JSON 파일 복원** (백업 파일 업로드)

### 2-2. 선수 등록 (랭킹 관리)
| 방법 | 경로 |
|------|------|
| 개별 등록 | 랭킹 → 선수 등록 버튼 → 이름·학교·부문·성별·포인트·등록번호·연락처 입력 |
| CSV 일괄 등록 | 랭킹 → CSV 가져오기 → 양식 다운로드 → 작성 후 업로드 → 중복 자동 감지 |
| 기존 데이터 복원 | 설정 → JSON 파일 복원 |

**CSV 양식**: `이름,학교,부문,성별,포인트`  
부문: `초등 / 중등 / 고등 / 대학 / 일반 / 생활체육`  
성별: `남 / 여`

### 2-3. 복식 페어 등록
랭킹 → 복식 탭 → 페어 등록 → 선수1·선수2 선택 → 종류(남복/여복/혼복) 선택

### 2-4. 단체팀 등록
랭킹 → 단체팀 탭 → 팀 등록 → 팀명·소속·부문·성별·구성원 선택

## 3. 대회 생성

**대회·대진표 → 새 대회 생성**

| 항목 | 설명 |
|------|------|
| 대회명 | 예: 2026 전국 초중고 탁구 선수권대회 |
| 날짜 / 장소 | 대회일·경기장명 |
| 대회 등급 | S급 / A급 / B급 / C급 / 생활체육S~B (포인트 배율에 영향) |
| 경기 방식 | 5세트제 11점 (기본) / 3세트제 / 7세트제 |

### 종목 추가
각 종목마다 → 부문·종류·성별·대진형식 선택

| 대진형식 | 적합 인원 | 특징 |
|---------|----------|------|
| 토너먼트 | 8~128명 | 단판 탈락, 시드 배정 |
| 리그 | 4~12명 | 전원 대전, 승점 순위 |
| 조별+토너먼트 | 16~64명 | 예선 조별 → 본선 토너먼트 |
| 시드예선 | 16~64명 | 상위 N명 본선 직행 + 나머지 예선 |
| 더블엘리미네이션 | 4·8·16·32명 | 승자조+패자조, 2패 시 탈락 (2의 거듭제곱) |

→ **대회 생성** 클릭 → 각 종목에서 **참가자 배정** → **대진표 생성**

## 4. 경기일정표 생성

**경기일정표 → 새 일정표 만들기**

1. 일정표 이름·날짜·시작시간 입력
2. 코트 수·경기당 시간(분)·버퍼 설정
3. **종목별 세부 설정** (경기 수·분·코트 수 자동 계산)
4. 다일차 대회: **일차 추가** 버튼
5. **일정표 생성** → 코트순·시간순 보기 → **CSV 내보내기** or **인쇄**

### 대회 병렬 시간표 자동생성 (★ 핵심)
일정표 상세 좌측 → **대회 병렬 시간표 자동생성** → 대회 선택 → **병렬 시간표 생성**
- 대진 구조를 분석해 **코트를 최대 병렬**로 채우고, **라운드 의존성**(이전 라운드 종료 후 다음 시작),
  **조별→본선** 순서, **같은 선수 동시경기 금지·휴식**을 지키며 **총 소요시간(makespan)을 최소화**.
- 결과: 코트·시간·선수가 채워진 일정표 + 대진표에 시간/코트 기재 + "N경기 병렬배치·총 X시간·가동률 Y%".
- 코트 수·시작시간·경기시간은 일정표의 종목 설정에서 자동 도출.

## 5. 대회 당일 운영

### 5-1. 체크인 (선수 입장 확인)
**체크인 → 체크인 스테이션**

- **QR 스캔**: QR 리더기로 선수증 스캔 (USB HID 방식, 별도 드라이버 불필요)
- **수동 검색**: 이름/소속/등록번호 검색 → 체크인 버튼
- 현황 탭: 부문별 체크인 진행률 확인
- QR 선수증 출력: QR 선수증 탭 → 인쇄

> **QR 선수증 사전 준비**: 대회 전날 QR 선수증 출력 → 선수에게 배포

### 5-2. 대시보드 열기
**운영 대시보드** → 실시간 시계 + 현황 표시  
→ 브라우저 알림 허용 (우상단 **알림 허용** 버튼 클릭)

### 5-3. 경기 호출 (콜링 시스템)
대시보드 → **경기 호출** 섹션
1. 대회 선택 → 종목 선택 → 경기 선택
2. 탁구대 번호 입력 → **호출** 클릭
3. → 라이브 보드에 즉시 표시 + 브라우저 알림 전송
4. 선수 입장 확인 후 → **확인** 클릭

### 5-4. 실시간 스코어 입력 (모바일 최적화)
**점수 입력 → 실시간 스코어** (주심 스마트폰 사용 권장)

1. 대회·종목·경기 선택 → 탁구대 번호 → **스코어보드 시작**
2. 파란(선수1)/빨간(선수2) 버튼 터치 → 점수 자동 집계
3. 서브 표시 자동 (2점마다 교체, 듀스 시 1점마다)
4. 세트 완료 시 자동 다음 세트 이동
5. 경기 종료 → 포인트/Elo 자동 반영 → 대진표 자동 업데이트

**취소(Undo)**: 점수 아래 ↩ 버튼

**키보드 단축키** (스코어보드 화면):
| 키 | 동작 |
|----|------|
| `[` | 선수1 점수 +1 |
| `]` | 선수2 점수 +1 |
| `z` | 선수1 마지막 점수 취소 |
| `x` | 선수2 마지막 점수 취소 |

### 5-5. 수동 점수 입력
**점수 입력 → 직접입력**

1. 대회·종목·경기 선택
2. 세트별 점수 입력 (Enter로 세트 추가)
3. 결과 미리보기 확인 → **결과 저장 & 포인트 반영**
4. 미확인 상태로 저장 → 대시보드에서 확인 처리

### 5-6. 라이브 보드 (TV/프로젝터)
**라이브 보드** 페이지 → **⛶ 전체화면** 클릭  
경기장 모니터/빔프로젝터에 띄워두면 자동 업데이트:
- 현재 진행중 경기 (실시간 점수)
- 경기 호출 목록
- 대기중 경기
- 최근 결과
- 종목별 진행률

## 7. 랭킹 관리

### 포인트 시스템
- 토너먼트 종료 시 승자→`pointsForWin`, 패자→`pointsForWin × 20%` 자동 적립
- **수동 조정**: 선수 행 → **+P** 버튼 → 포인트 입력 → 승/패 체크

### Elo 레이팅 (USATT 방식)
- 초기 레이팅: 포인트 기반 자동 환산 (1000~2800)
- 경기마다 자동 갱신 (K-factor: 초보 32, 중급 24, 고수 16)
- 등급 표시: Beginner / Developing / Intermediate / Advanced / Expert / Elite

### 선수 전적 조회
랭킹 → 선수 행 → 📊 아이콘 → 대회 경기 이력 + 점수 기록 + 승률 확인

### 내보내기
| 버튼 | 내용 |
|------|------|
| CSV 내보내기 | 이름·학교·부문·성별·포인트·Elo·등록번호·연락처 |
| CSV 가져오기 | 양식 기반 일괄 등록 (중복 자동 건너뜀) |

## 12-6. v3.8 (이번 세션) — 종합 운영 감사 7×7 + 정확성 수정

분석팀 7명 + 검증단 7명으로 시드~대진~정산~일정~점수~운영~데이터 전 도메인을 교차감사하고,
오탐을 걸러낸 실제 버그·갭만 수정.

### 확정 수정 (1차)
- **조별 진출 클램프**: `advanceCount > 조원수` 시 채워지지 않는 본선 고아 슬롯이 생겨 종목이
  영구 미완료되던 것 → 실제 조원 수로 제한(generateKnockoutFromGroups).
- **듀스 서브 표시**: 듀스(양쪽 10점+)에서 서브 교대가 2점 간격으로 잘못 표시 → 1점 간격 수정.
- **세트 점수 검증**: 직접입력에서 비정상 세트(예 15:8) 앰버 경고(isValidSetScore).
- **백업 보완**: liveMatches·matchCalls 백업/복원 포함, 버전 라벨 정리.

### 확정 수정 (2차)
- **직접대결 동률**: 리그/조별 순위 동률 시 직접대결(head-to-head)→초기순서로 결정적 정렬
  (standingsComparator). 삼각 동률에서도 순위 안정.
- **일정 초과 경고**: 자동 일정생성 시 운영시간 초과로 배치되지 못한 경기 수를 생성 시 고지.
- **미체크인 경고**: 대시보드 대기경기에 체크인 안 한 선수 경고 배지.
- **빈 코트 제안**: 경기 호출에 비어 있는 코트 번호 빠른 설정 버튼.

### 확정 수정 (3차)
- **조별→본선 배선 idempotent 재구성**(slotRef): 조 결과를 사후 수정해 진출자가 바뀌면
  본선 대진이 **자동 갱신**됨. 부전승 본선 슬롯도 실선수로 정확히 해석. 단위테스트 7개.
- **코트별 분리 인쇄**: 코트순 보기에서 '코트별' 버튼 → 탁구대마다 일정표 1장씩 출력(현장 배부).

### 검증으로 걸러낸 오탐(수정 불필요)
부전승 loserId(이미 정상), 시드예선 qualAdvance(이미 클램프), 대회 조기완료(bye는 !isBye로 제외).

### 의도적 보류 (설계 결정)
- 복식·단체전은 포인트·승패만 반영되고 개별 Elo는 미반영(종목 배율 0.6~0.8배로 부분 보정).
  검증단 권고에 따라 정산엔진 안정성 우선으로 보류.
- 일정 라운드 의존 자동조정·드래그 편집은 향후 개선 후보.

## 12-3. v3.5 추가 기능 (이전 세션) — 토너먼트 운영 정확도 재구성

### 포인트·Elo 정산 모델 전면 교체 (경기별 즉시 가산 → 종목 완료 시 일괄 정산)

  - `computeEventPlacements(ev)`: 브래킷에서 최종 성적(우승/준우승/3위/4위/8강/16강…/참가) 산정

### 고친 포인트 계산 버그 (v3.4 잔존)
- 결승 패자가 **우승 포인트**를 받던 것 → 준우승 포인트로 정정

- 조별/리그 패자가 **매 패배마다 참가 포인트** 중복 → 성적 기반 1회

### 대진표 안정성
- **결과 취소 연쇄 정리**(`propagateAndCascade`): 상위 라운드 결과를 취소하면

  (단, v3.4 이전에 생성된 진행 중 시드예선 대진은 재생성해야 적용됨)

- `TournamentDetail`이 스코프에 없는 `syncStatus`/`syncTournament`를 참조 → **env 설정 시
